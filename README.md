# Unicode 查询

> Unicode 字符查询工具，编解码，UTF-8/UTF-16 显示，常用字符表

## 功能概览

- **字符到 Unicode 码点互查：输入字符显示码点，或输入码点显示字符**
- **UTF-8 / UTF-16 编码字节序列详细显示**
- **UTF-8 编码器：任意文本转为 UTF-8 字节数组**
- **常用字符表：Unicode 区块分类浏览（CJK/拉丁/希腊/箭头等）**
- **码点复制：U+XXXX 格式与 HTML/JS Unicode 转义格式**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
